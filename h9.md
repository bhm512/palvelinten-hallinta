# h9

Tehtävänanto: Korvaava tehtävä tunnista: esittele, kuinka käytät Git:iä jonkun muun palvelimen kuin Githubin kanssa. Voit käyttää esimerkiksi omaa ssh-palvelinta tai palveluntarjoajaa, kuten Gitlab. Tee varastosta kaksi paikallista kloonia (clone) ja näytä, että muutokset päivittyvät kloonista toiseen. Esitä kaikki vaiheet, ota mukaan ruutukaappauksia. Etsi tarvittaessa itse lähteitä ja ohjeita. Palauta kohtaan h9.

Kohta 1: Ympäristöstä

Palvelin: Debian OS
Isäntäkone: Mac OS (Mac Book Air M2)
Yhteys palvelimeen: SSH-avainautentikointi
Git-versio: 2.47.3

Tehtävä: 
Ensiksi loin sudo adduser git komennolla erillisen käyttäjän, joka on Git operaatioita varten.


Seuraavaksi loin SSH-avaimen Macille seuraavanlaisesti komennolla: ssh-keygen -t ed25519
